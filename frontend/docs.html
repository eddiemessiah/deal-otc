@defimessiah/dealotc-sdk

The Deal OTC API and Node.js SDK allows enterprise clients, family offices, and partner exchanges to programmatically onboard clients, parse compliance documents, and retrieve Walled Garden Readiness scores.

npm install @defimessiah/dealotc-sdk

1. Initialize Client

Authenticate using your Deal OTC Enterprise API key.

const { DealOTC } = require('@defimessiah/dealotc-sdk');

const dealotc = new DealOTC({ 
    apiKey: 'sk_live_12345...',
    environment: 'production'
});

2. Automate Document Parsing (RAG Engine)

Upload a mandate letter or proof of funds. Our AI clause extractor parses the document and updates the CRM.

const parseResult = await dealotc.compliance.parseDocument(
    "123_APEX_FAMILY_OFFICE", 
    'proof_of_funds', 
    pdfBuffer
);

console.log(parseResult.analysis);
// Output: 
// { 
//   approved: true, 
//   extractedCapital: "$15,000,000 USD", 
//   jurisdiction: "Dubai",
//   redFlags: [] 
// }

3. Check Readiness Score & Automated CRM Email Actions

Retrieve the client's current status in the 3-Tier Walled Garden. If documents are missing, our backend triggers SendGrid/Resend to automatically email the client requesting the exact missing PDF.

const status = await dealotc.client.getStatus("123_APEX_FAMILY_OFFICE");

if (status.score < 90) {
    console.log(`Missing Docs: ${status.missingDocuments.join(', ')}`);
    // Deal OTC backend automatically emails client: "Please upload your UBO_Registry"
} else {
    const report = await dealotc.reports.generateProgramOfOperations("123_APEX_FAMILY_OFFICE");
    console.log("Client is Ready to Trade.");
}
